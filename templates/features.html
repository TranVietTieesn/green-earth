💨 {% if lang == 'en' %}Perfect Atmosphere{% elif lang == 'ja' %}完璧な大気{% else %}Khí quyển hoàn hảo{% endif %}

{% if lang == 'en' %} Earth's atmosphere contains 21% oxygen and 78% nitrogen, creating the perfect balance for life. The ozone layer protects us from harmful UV radiation while greenhouse gases keep our planet warm. {% elif lang == 'ja' %} 地球の大気は21%の酸素と78%の窒素を含み、生命にとって完璧なバランスを作り出しています。 オゾン層は有害な紫外線から私たちを守り、温室効果ガスは私たちの惑星を温かく保っています。 {% else %} Khí quyển Trái Đất chứa 21% oxy và 78% nitơ, tạo ra sự cân bằng hoàn hảo cho sự sống. Tầng ozon bảo vệ chúng ta khỏi bức xạ UV có hại trong khi các khí nhà kính giữ cho hành tinh của chúng ta ấm áp. {% endif %}

🌡️ {% if lang == 'en' %}Goldilocks Zone{% elif lang == 'ja' %}ゴルディロックスゾーン{% else %}Vùng Goldilocks{% endif %}

{% if lang == 'en' %} Earth orbits at the perfect distance from the Sun - not too hot, not too cold - allowing liquid water to exist. This habitable zone is crucial for sustaining life as we know it. {% elif lang == 'ja' %} 地球は太陽から完璧な距離を軌道に乗っています - 暑すぎず、寒すぎず - 液体の水が存在することを可能にしています。 この居住可能ゾーンは、私たちが知っている生命を維持するために不可欠です。 {% else %} Trái Đất quay quanh quỹ đạo ở khoảng cách hoàn hảo từ Mặt trời - không quá nóng, không quá lạnh - cho phép nước lỏng tồn tại. Vùng có thể sinh sống này rất quan trọng để duy trì sự sống như chúng ta biết. {% endif %}

🧲 {% if lang == 'en' %}Protective Magnetosphere{% elif lang == 'ja' %}保護的磁気圏{% else %}Từ quyển bảo vệ{% endif %}

{% if lang == 'en' %} Earth's magnetic field deflects harmful solar wind and cosmic radiation, creating beautiful auroras while protecting our atmosphere from being stripped away like on Mars. {% elif lang == 'ja' %} 地球の磁場は有害な太陽風と宇宙放射線を逸らし、美しいオーロラを作り出しながら、 火星のように大気が剥ぎ取られることから保護しています。 {% else %} Từ trường Trái Đất làm lệch hướng gió mặt trời có hại và bức xạ vũ trụ, tạo ra cực quang tuyệt đẹp trong khi bảo vệ khí quyển của chúng ta khỏi bị tước đi như trên sao Hỏa. {% endif %}

{% if lang == 'en' %}Amazing Natural Phenomena{% elif lang == 'ja' %}驚くべき自然現象{% else %}Hiện tượng tự nhiên tuyệt vời{% endif %}

{% if lang == 'en' %}Weather Systems{% elif lang == 'ja' %}気象システム{% else %}Hệ thống thời tiết{% endif %}

{% if lang == 'en' %} From hurricanes and tornadoes to rainbows and lightning, Earth's dynamic weather creates spectacular displays of natural power {% elif lang == 'ja' %} ハリケーンや竜巻から虹や稲妻まで、地球のダイナミックな天気は自然の力の壮観な表示を作り出します {% else %} Từ bão và lốc xoáy đến cầu vồng và sét, thời tiết năng động của Trái Đất tạo ra những màn trình diễn ngoạn mục của sức mạnh tự nhiên {% endif %}

🌋

{% if lang == 'en' %}Volcanic Activity{% elif lang == 'ja' %}火山活動{% else %}Hoạt động núi lửa{% endif %}

{% if lang == 'en' %} About 1,500 active volcanoes shape our planet's surface and create new land while recycling materials from deep within Earth {% elif lang == 'ja' %} 約1,500の活火山が私たちの惑星の表面を形作り、地球の深部からの物質をリサイクルしながら新しい土地を作り出しています {% else %} Khoảng 1.500 núi lửa hoạt động định hình bề mặt hành tinh và tạo ra đất mới trong khi tái chế vật liệu từ sâu bên trong Trái Đất {% endif %}

🌊

{% if lang == 'en' %}Ocean Currents{% elif lang == 'ja' %}海流{% else %}Dòng hải lưu{% endif %}

{% if lang == 'en' %} Massive ocean currents transport heat around the globe, regulating climate and supporting marine ecosystems worldwide {% elif lang == 'ja' %} 巨大な海流が地球全体に熱を運び、気候を調節し、世界中の海洋生態系を支えています {% else %} Các dòng hải lưu khổng lồ vận chuyển nhiệt quanh địa cầu, điều hòa khí hậu và hỗ trợ các hệ sinh thái biển trên toàn thế giới {% endif %}

⛰️

{% if lang == 'en' %}Tectonic Plates{% elif lang == 'ja' %}プレートテクトニクス{% else %}Mảng kiến tạo{% endif %}

{% if lang == 'en' %} Earth's moving crustal plates create mountains, valleys, and ocean basins while causing earthquakes and volcanic eruptions {% elif lang == 'ja' %} 地球の移動する地殻プレートは山、谷、海盆を作り出し、地震や火山噴火を引き起こします {% else %} Các mảng vỏ Trái Đất di chuyển tạo ra núi, thung lũng và lưu vực đại dương trong khi gây ra động đất và phun trào núi lửa {% endif %}

{% if lang == 'en' %}Seasonal Cycles{% elif lang == 'ja' %}季節の循環{% else %}Chu kỳ mùa{% endif %}

{% if lang == 'en' %} Earth's 23.5-degree axial tilt creates our beautiful seasonal changes. As our planet orbits the Sun, different regions receive varying amounts of sunlight throughout the year, resulting in spring's rebirth, summer's abundance, autumn's harvest, and winter's rest. This natural rhythm has shaped life on Earth for millions of years. {% elif lang == 'ja' %} 地球の23.5度の軸の傾きが美しい季節の変化を作り出しています。私たちの惑星が太陽の周りを軌道に乗ると、 異なる地域が年間を通じて様々な量の日光を受け、春の再生、夏の豊富さ、秋の収穫、冬の休息をもたらします。 この自然のリズムは何百万年もの間、地球上の生命を形作ってきました。 {% else %} Độ nghiêng trục 23,5 độ của Trái Đất tạo ra những thay đổi mùa tuyệt đẹp. Khi hành tinh của chúng ta quay quanh Mặt trời, các vùng khác nhau nhận được lượng ánh sáng mặt trời khác nhau trong suốt năm, dẫn đến sự tái sinh của mùa xuân, sự phong phú của mùa hè, mùa thu thu hoạch và mùa đông nghỉ ngơi. Nhịp điệu tự nhiên này đã định hình sự sống trên Trái Đất trong hàng triệu năm. {% endif %}
